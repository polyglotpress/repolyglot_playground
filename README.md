How to run:

Download and open main folder in vs code.
Make sure you have Node installed on your PC device.
Run the following line in terminal:
npm init
Install dependencies:
npm install express ejs-mate mongoose path dotenv
create a file in the main folder, name it ".env".
Add your MongoDB URL to a variable called MONGO_URI

run in terminal:
node server.js

on your browser, paste the following:
localhost:5000
